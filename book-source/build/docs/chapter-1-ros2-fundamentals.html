What's NOT in This Chapterâ€‹

We're keeping it focused on software concepts only:

  • No hardware (Turtlesim is software)
  • No physics simulation (that's Chapter 2: Gazebo)
  • No sensors yet (Chapter 3 covers sensors)
  • No machine learning (later chapters)
  • No autonomous movement yet (we control the turtle explicitly)

This focus keeps you from being overwhelmed. Master ROS 2 basics first, then add complexity.

How to Use This Chapterâ€‹

  1. Read sequentially. Each lesson builds on the previous one.
  2. Type the code yourself (don't copy-paste). This builds muscle memory.
  3. Ask Claude questions. Every lesson has prompts like "Ask Claude: Why is this topic named with a slash?"
  4. Experiment. Change values in code, see what breaks, fix it.
  5. Troubleshooting first. If something fails, check the Troubleshooting section in the lesson before asking for help.
